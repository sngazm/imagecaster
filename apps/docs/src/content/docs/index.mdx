Cloudflare のインフラを使った、セルフホスト型 Podcast 配信ツールキットです。音声・メタデータ・RSS フィードはすべて R2 上の静的ファイルとして保存されます。データベースも状態も持ちません。

## はじめる

- [概要](getting-started/overview/) — システムの全体像と設計の考え方
- [セットアップ](getting-started/setup/) — 一からの構築手順

## システム構成

| コンポーネント | 技術 | 役割 |
|---|---|---|
| API | Cloudflare Workers + Hono | エピソード管理・アップロード処理 |
| 公開サイト | Astro SSG | リスナー向け静的サイト |
| 管理画面 | React + Vite | エピソード・設定の管理 |
| ストレージ | Cloudflare R2 | 音声・メタデータ・RSS フィード |
| 認証 | Cloudflare Access | ゼロトラスト認証 |
