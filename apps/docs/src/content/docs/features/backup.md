管理画面からすべてのデータをバックアップ（ZIP ファイル）としてエクスポート・インポートできます。

## エクスポート

管理画面の **設定** → **バックアップ** からエクスポートできます。

エクスポートされる ZIP ファイルには以下が含まれます:

```
backup-YYYY-MM-DD/
├── index.json         # Podcast インデックス（設定 + エピソード一覧）
├── templates.json     # 概要欄テンプレート
└── episodes/
    └── {storageKey}/
        └── meta.json  # 各エピソードのメタデータ
```

音声ファイル自体はバックアップに含まれません（R2 に保存済みのため）。

## インポート

エクスポートした ZIP ファイルを管理画面からインポートできます。

インポート時の動作:
- 既存のエピソードは上書きされます
- R2 の音声ファイルはそのまま残ります
- `index.json` と `templates.json` は完全に上書きされます

## API

バックアップ機能は以下の API エンドポイントで提供されます。

| メソッド | パス | 説明 |
|--------|------|------|
| `GET` | `/api/backup/export` | バックアップデータ取得 |
| `POST` | `/api/backup/import` | バックアップデータ復元 |
